## Git 指令說明

1. 說明 git clone, git pull, git checkout 差異。
git clone:簡單初始化方式 當你已經有一個git程式庫 只需要在本地clone 一份
git pull: 從遠端更新到本端
git checkout: 切換分支

2. 說明 git commit, git push 差異。
git push: 將 Commit 送出去 更新到遠端
git commit : 提交當前修改內容 提交時必須用-m 來輸入訊息
